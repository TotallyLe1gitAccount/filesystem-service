# 📁 FileSystem Service

Учебный проект на Python для работы с файлами (текст, бинарные данные, сериализация).

---

# 🧠 Возможности

- 📄 Работа с текстовыми файлами
- 🧬 Работа с бинарными файлами
- 💾 Сериализация / десериализация (`pickle`)
- 🔧 Проверка и удаление файлов
- 📊 Чтение файла целиком и по частям

---

# 🏗️ Структура
services/
├── base.py # базовый сервис
├── file.py # текстовые файлы
├── binary.py # бинарные файлы
├── serialization.py # pickle сериализация
└── constraints.py # режимы файлов

⚙️ Требования
- Python 3.10+
- только стандартная библиотека
